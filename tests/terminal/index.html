协议信息

协议版本: JSON + Base64 (参考 1Panel)
消息格式:
  客户端 → 服务端 (JSON):
    {"type":"cmd","data":"base64..."}      = 输入
    {"type":"resize","cols":80,"rows":24}  = 调整大小
    {"type":"heartbeat","timestamp":123}   = 心跳

  服务端 → 客户端 (JSON):
    {"type":"cmd","data":"base64..."}      = 输出
    {"type":"heartbeat","timestamp":123}   = 心跳响应

特性:
  JSON 协议 + Base64 编码
  多客户端共享终端 (N:1)
  心跳保活机制 (10秒)
  动态调整窗口大小
  历史缓存和持久化 (10MB 环形缓冲区)
  PTY 进程状态监控
  会话和进程状态分离
